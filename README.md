# Thesis Benchmarking Project

Repository foundation for a Bachelor's thesis experiment platform that compares selected neuro-symbolic AI architecture families under one evaluation protocol.

Current scope:
- repository structure
- architecture notes
- progress log
- configuration placeholders
- environment setup files
- dataset infrastructure for MNLogic-first development
- shared encoder and common model foundations

Not implemented yet:
- model training
- evaluation logic
- API or UI behavior

- `docs/architecture.md`

- `docs/dataset-setup.md`
- `docs/model-foundations.md`
