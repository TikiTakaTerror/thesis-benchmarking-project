# Thesis Benchmarking Project

Phase 0 repository skeleton for a Bachelor's thesis experiment platform that compares selected neuro-symbolic AI architecture families under one evaluation protocol.

Current scope:
- repository structure
- architecture notes
- progress log
- configuration placeholders
- environment setup files
- dataset infrastructure for MNLogic-first development

Not implemented yet:
- model training
- evaluation logic
- API or UI behavior

Key docs:

- `docs/environment-setup.md`
- `docs/dataset-setup.md`
